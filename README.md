## 📌 Project Overview

The Mobile Sales Analytics Dashboard is an interactive business intelligence solution developed using Microsoft Power BI. It provides comprehensive insights into mobile sales performance by analyzing revenue, transactions, customer ratings, payment methods, product performance, and sales trends.

The dashboard enables business users to monitor key performance indicators and make informed business decisions through interactive visualizations and filters.

---

## 🎯 Business Objective

The objective of this dashboard is to help stakeholders:

- Monitor overall sales performance
- Analyze sales by mobile brand and model
- Evaluate customer ratings
- Track monthly sales trends
- Compare payment methods
- Analyze regional sales performance
- Support data-driven decision making
